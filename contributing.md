# Contributing

Thank you for contributing to the **Awesome ML Conference Paper Writing Style Guide**.

## What to Contribute
- Add or update conference guidelines (NeurIPS, ICML, ICLR, ACL, etc.)
- Fix errors or outdated information
- Improve clarity, structure, or examples (LaTeX, citations)

## Guidelines
- Use **accurate, verifiable, official sources**
- Keep content **concise, structured, and consistent**
- Follow existing README structure and formatting
- Ensure links are **working**
